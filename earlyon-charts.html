This page is a work in progress :) The data you are currently viewing covers the period from December 1st, 2023 to December 31st, 2023.

Please note that there were 5 Fridays and 5 Saturdays in December 2023.

Based on the data we've accumulated so far, we can make the following observations:

  • There are a lot less Saturday playgroups in the "South" area than other places
  • There are never any playgroups anywhere in the city on Sundays
  • There are no baby playgroups on Saturdays
